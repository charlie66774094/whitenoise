# 白噪音小站 - 你的随身静谧空间
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
> 一个简约而温暖的白噪音网站，带你回归内心的宁静。
## 🌟 网站简介
欢迎来到 **白噪音小站**，这是一个由热爱自然与宁静的开发者精心打造的在线音频空间。在这里，你可以通过简单的点击，聆听纯净的粉红噪音（Pink Noise），让轻柔的声音如微风般拂过耳畔，帮助你放松身心、专注工作、改善睡眠，或是逃离喧嚣，短暂地沉浸于一片静谧之中。
本项目以极简设计为核心，无需下载、无需注册，打开即用，为你提供最直接、最纯粹的声音陪伴。
## 🎧 功能特色
- **纯净粉红噪音**：采用高质量音频生成算法，模拟自然界中舒适的粉红噪音，比传统白噪音更柔和、更接近雨声或溪流声。
- **一键播放/暂停**：界面简洁直观，点击中心按钮即可控制声音开关。
- **持续生成**：声音实时生成，不依赖音频文件，节省流量，保护隐私。
- **响应式设计**：适配手机、平板与桌面设备，随时随地享受宁静。
- **无广告、无追踪**：完全免费且无任何商业干扰，尊重你的每一次访问。
